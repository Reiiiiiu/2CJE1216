### 作品説明
以上のソースコードはPythonのライブラリーNumpyで作成した簡単な2048ゲームです。

### 2048のルールを説明
4×4のマスに数字が書かれたタイルがあり、スライドさせるとそれらはマスの端まで移動し、同時に新たなタイル(90%の確率で２のタイル､10%の確率で４のタイル)が出現する。 同じ数字のタイルがぶつかると2+2=4、4+4=8というように数字が足し合わされていく。
